# 项目跟进助手 — HTML 设计稿方案

## 目标
输出一个独立 `project-tracker.html` 文件，无外部依赖（仅 CDN 引入 Chart.js 用于图表），内嵌完整模拟数据，可在浏览器中直接预览并完整交互。

---

## 设计系统

**视觉风格**
- 整体采用深色主题（背景 `#0A0E1A`），营造专业感
- 主强调色：靛蓝 `#4F6EF7`；危险：`#F75E5E`；警告：`#F7B94F`；成功：`#3ECF8E`
- 字体：`-apple-system, "PingFang SC", "Microsoft YaHei"` 系统字体栈，中文优先
- 卡片使用半透明背景 + 细边框（`rgba(255,255,255,0.06)`），无阴影，扁平化

**交互规范**
- 左侧固定导航栏 + 右侧内容区三栏布局
- 视图切换平滑过渡（CSS transition，无重加载感）
- 项目卡片点击展开详情抽屉（右侧滑入面板）
- FMEA 表格行内编辑（模拟态）

## 核心模块

### 模块 1 — 全局仪表板（Dashboard）
- **顶部 KPI 栏**：4 个 Stat 卡（总项目数 10 / 正常推进 / 存在风险 / 延期预警）
- **项目健康矩阵**：10 个项目卡片，Grid 排列，每张卡显示：
  - 项目名称 + 负责人 + 到期日
  - 圆环进度（SVG 手绘，纯 CSS 动画）
  - 健康状态标签（绿/黄/红）
  - 本周最新动态一句话
- **风险全景热力图**：横轴=项目，纵轴=风险等级，色块显示当前风险聚集

### 模块 2 — 项目详情抽屉（点击任意项目卡后右滑出现）
- **基础信息**：里程碑节点时间线（横向滚动，含状态标记）
- **任务进展**：分组任务列表（已完成/进行中/未开始），带进度条
- **风险清单**：该项目的 FMEA 风险条目预览，点击跳转完整分析
- **跟进事项**：Next Action 列表，负责人 + 截止日 + 优先级标签

### 模块 3 — FMEA 风险分析（独立页面）
FMEA（失效模式与影响分析）是制造业/项目管理经典风险工具：

- **RPN 评分矩阵**：每条风险 = 严重度(S) × 发生率(O) × 可探测度(D)，RPN = S×O×D
- **风险登记表**：可按项目筛选，列包含：
  - 功能/流程节点
  - 潜在失效模式
  - 失效影响
  - S / O / D 评分（1-10 滑块）
  - RPN（自动计算，高于 200 标红）
  - 预防/控制措施
  - 责任人 + 完成日期
- **风险优先排序图**：横向条形图，按 RPN 降序，高风险条目置顶

### 模块 4 — 跟进中心（Action Center）
- 卡片式 Kanban：待确认 / 跟进中 / 已关闭 三列
- 每个 Action 卡含：来源项目 badge + 优先级 + 负责人头像 + 截止日倒计时
- 本周重点跟进事项置顶高亮区

### 模块 5 — 数据导入区（设计态模拟）
- 路径输入框 + "读取文件"按钮（展示 UI，不实际读取）
- 已加载文件列表：显示文件名、工作表、最后刷新时间
- 字段映射说明（展示 Excel 列名 → 系统字段对照）

---

## 数据模型（内嵌模拟）

10 个项目示例，每个包含：
- 基础属性：id, name, owner, startDate, dueDate, progress(%)
- 健康状态：`healthy` / `at_risk` / `delayed`
- 里程碑数组（含状态）
- FMEA 风险条目数组（每项目 3-5 条）
- 跟进事项数组

---

## 技术实现

- **单 HTML 文件**，`<style>` 内嵌 CSS，`<script>` 内嵌 JS
- **Chart.js CDN**（仅用于风险 RPN 条形图，轻量依赖）
- SVG 圆环进度：纯代码生成，无图片资源
- 交互：原生 JS DOM 操作，无框架
- 响应式：支持 1280px 以上宽屏
